Welcome to the Game of Set

In this game, you need to find the three cards on the table that make up a set. Each card has four attributes: color, number, shape, and shading. For example, Take a look at this card:

placeholder

This card has red coloring, a number of 1, a shape of hexagon, and vertical shading. Three cards make up a set when, for each attribute, they all have different values or the same values.

These three cards make up a set:
            All three cards have the same color: red
            All three cards have different shapes: hexagon, triangle, and square
            All three cards have different numbers: 1, 0, and 2
            All three cards have the same shading: vertical
placeholder

These three cards do not make up a set. This is because two of the cards have the color red while only one has a different color (blue). In the case where there is no set currently on the table, select the NoSet option to add three more cards. Each correctly identified set adds one point to the player's score. Each incorrect guess at a set takes one point away from the player's score. The goal of the game is to have the highest score once the deck completely runs out of cards.
