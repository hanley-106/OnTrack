Game of Set

Tutorial

In this game, you need to find the three cards on the table that make up a set. Each card has four attributes: color, number, shape, and shading. For example, Take a look at this card:

The above card has red coloring, a number of 1, a shape of oval, and striped shading. Three cards make up a set when, for each attribute, they all have different values or the same values. 

These three cards make up a set:
            All three cards have the same color: red.
            All three cards have different shapes: oval, square, and almond.
            All three cards have different numbers: 2, 1, and 3.
            All three cards have the same shading: striped.

These three cards above do not make up a set. This is because two of the cards have the color red while only one has a different color (purple). In the case where there is no set currently on the table, select the "No Set" option to add three more cards. Each correctly identified set adds one point to the player's score. Each incorrect guess at a set takes one point away from the player's score. The goal of the game is to have the highest score once the deck completely runs out of cards.

How Scoring Works: Single Player

The player and computer both start at 0. Both the player and computer take alternating turns. If the player:

  • Find a set: +3
  • Click No set present button but set present: -1
  • Click no set present, no set present: +0
  • Use hint, subtracts 1 from score only once every card change: -1
If the computer:
  • Find a set: +3
  • No set found: -1

How Scoring Works: Multi Player

Both players start at 0. Both the players. Whenever it is the players turn, they have to click their name. Player scoring:

  • Find a set: +3
  • Click No set present button but set present: -1
  • Click no set present, no set present: +0
  • Use hint, subtracts 1 from score only once every card change: -1

How To Unselect Cards

Click on the card again and the red border around it will go away. This means that you have unselected the card.
